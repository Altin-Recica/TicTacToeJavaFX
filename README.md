# Boter Kaas Eieren met JavaFX

Een simpele **Tic-Tac-Toe** game gemaakt met **JavaFX**. Twee spelers kunnen om de beurt X of O zetten en proberen drie op een rij te krijgen.

## Functies

- 1v1 lokaal of 1vsAI 
- Win- en gelijkspel detectie
- Gemakkelijk te resetten en opnieuw te spelen
- Simpele en duidelijke GUI

## Hoe te gebruiken

1. Clone de repo:  
   ```bash
     git clone https://github.com/Altin-Recica/BoterKaasEierenJavaFX.git
   ```
2. Open het project in je IDE.
3. Voeg de JavaFX bibliotheek toe.
4. Run Main.java en begin met spelen!
